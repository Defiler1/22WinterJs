# 1. 코드 작성 규칙
- 세미콜론: 생략가능하나 문장끝에 추가 추천
- 주석 (comment)
  - 한줄주석: //
  - 여러줄주석: /* */

# 자료형(Data Type)

- 문자열(String)
  - ' '
  - " "
    - Kim's family
    - -> "Kim's family"
  - ``(백틱)
    - 템플릿 문자열
    - let i = 100; `${i}값 출력함`
  
  
- typeof: 자료형을 알려주는 연산자
  - typeof 데이터(값-리터럴, 변수값)
  
- escape squence
  - Kim's family
  - -> Kim\\'s family
